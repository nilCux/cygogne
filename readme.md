## Mongo DB
* Installlation: https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-ubuntu/
* Launch
> sudo systemctl start mongod
* Check status
> sudo systemctl status mongod
* Stop
> sudo systemctl stop mongod
* Restart
> sudo systemctl restart mongod
